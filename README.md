# Wordly+

Wordly+ is a single-page dictionary application built using HTML, CSS, and JavaScript.

Users can search for words and view:
- Definitions
- Pronunciation
- Example sentences
- Synonyms
- Pronunciation audio

The application also includes:
- Search history
- Saved favorite words
- Random word generator
- Light/Dark mode toggle
- Responsive design

---

# Features

## Word Search
Users can search for words using the search bar.

## Dictionary API
The app fetches live dictionary data from:

https://dictionaryapi.dev/

## Pronunciation Audio
Users can listen to the pronunciation of supported words.

## Synonym Tags
Synonyms appear as clickable tags that automatically perform a new search.

## Search History
Recently searched words are stored and displayed on the page.

## Saved Words
Users can save favorite words using localStorage.

## Theme Toggle
Users can switch between dark mode and light mode.

## Responsive Design
The layout adjusts for smaller screen sizes using media queries.
